# Task 005: Session Page UI

## Goal

Build the session page with video call UI: local and remote video feeds, basic layout.

## Deliverables

- [ ] `src/pages/` or `src/app/` — session page
- [ ] Video call view: tutor + student feeds (local + remote)
- [ ] Wire frame sampler to both video elements
- [ ] Basic styling (CSS/Tailwind) — minimal, readable

## Notes

- Role assignment: local = tutor, remote = student (or configurable)
- Graceful handling when camera denied or remote participant disconnects

## Verification

- Two tabs join; both see each other
- Frame sampler active for both streams
- UI is functional and readable
